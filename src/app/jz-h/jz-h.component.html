Welcome to the Jake Zur (jz) Command Line Interface (CLI). jz is a helpful command line tool for exploring Jake Zur's personal projects. This application is a terminal themed interface and supports the jz CLI. Use the terminal by typing a command and pressing 'enter'. You can move back to previous commands with the up arrow, and back with the down arrow. The following commands for the jz CLI are supported:

jz CLI supported commands:

  • jz -h -> display help message explaining package use and terminal navigation
  • jz lp -> lists all project names
  • jz ep <project_name> -> displays information about <project_name>
