Welcome to the Jake Zur (jz) Command Line Interface (CLI). jz is a helpful command line tool for exploring Jake Zur's personal projects. This application is a terminal themed interface that allows for navigating a users file directory via a Tree DS, and supports the the jz CLI. Use the terminal by typing a command and pressing 'enter'. The following commands for the jz CLI and file/folder navigation are supported:

jz CLI supported commands:

  • jz -h -> display help message explaining package use and terminal navigation
  • jz lp -> lists all project names
  • jz ep <project_name> -> displays information about <project_name>

Terminal navigation commands (coming soon!!):

  • mkdir <directory_name> -> creates a directory in the current working directory named <directory_name>
  • touch <file_name> -> creates a file in the current working directory named <file_name>
  • cd <directory_name> -> changes the current working directory to <directory_name>
